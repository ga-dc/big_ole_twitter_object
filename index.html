Exercise: Big Ol' Twitter Object

Check out app.js. This is a JSON response of a tweet from the twitter api. Answer the following questions and list the expression used to retrieve the value and answer the question.

Questions

  • Does this tweet have any favorites? "favourites_count": 24
  • When was is created? "created_at": "Wed May 23 06:01:13 +0000 2007"
  • How many URL's does this tweet have? "urls" 3
  • What is its short url? "url": "http://dev.twitter.com"
  • Which hashtags, if any are used? Twitterbird
  • What is the name of the person who tweeted this? "name": "Twitter API"
  • Do you follow them? "follow_request_sent": false
  • What is the URL of their website? "url": "http://dev.twitter.com"
  • How many tweets do they have? "listed_count": 10774
